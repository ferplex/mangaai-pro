# 🎨 MangaAI Backend

Backend API para **MangaAI Pro** - Generador de imágenes de anime/manga con Inteligencia Artificial usando Stable Diffusion.

## 🚀 Características

- ✅ Generación de imágenes desde texto (Text-to-Image)
- ✅ Soporte para múltiples estilos artísticos (Manga, Anime, Acuarela, etc.)
- ✅ API RESTful con CORS habilitado
- ✅ Integración con Stable Diffusion API
- ✅ Personalización de parámetros (CFG, seed, tamaño, etc.)

## 🛠️ Tecnologías

- **Node.js** - Runtime
- **Express** - Framework web
- **Node Fetch** - Cliente HTTP
- **CORS** - Middleware para peticiones cross-origin
- **Stable Diffusion API** - Motor de IA
